字蛛—中文WebFont压缩器

让网页自由嵌入中文字体成为可能

特性

  • 1. 轻巧：数 MB 的中文字体可被压成几十 KB
  • 2. 简单：完全基于 CSS，无需 js 与服务端支持
  • 3. 兼容：自动转码，支持 IE 与标准化的浏览器
  • 4. 自然：文本支持选中、搜索、翻译、朗读、缩放

原理

  • 1. 构建 CSS 语法树，提取字体路径与选择器规则
  • 2. 使用包含 WebFont 的 CSS 选择器索引站点的文本
  • 3. 删除 WebFont 中没有使用到的字符
  • 4. 转码成跨浏览器使用的字体格式

安装

先安装好 NodeJS，然后执行：

npm install font-spider -g

- Windows 下运行需要安装 perl 才能运行

- GruntJS 用户可使用 gruntjs 插件

使用

在 CSS 中使用 WebFont：

/*声明 WebFont*/
@font-face {
  font-family: 'pinghei';
  src: url('../font/pinghei.eot');
  src:
    url('../font/pinghei.eot?#font-spider') format('embedded-opentype'),
    url('../font/pinghei.woff') format('woff') 
    url('../font/pinghei.ttf') format('truetype'),
    url('../font/pinghei.svg') format('svg');
  font-weight: normal;
  font-style: normal;
}

/*使用选择器指定字体*/
.home h1, .demo > .test {
    font-family: 'pinghei';
}

1. @font-face 中的 src 定义的 .ttf 文件必须存在，其余的格式将由工具自动生成

2. 开发阶段请使用相对路径的 CSS 与 WebFont

运行 font-spider 命令：

font-spider ./demo/*.html
